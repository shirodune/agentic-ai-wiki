A concise, source-linked wiki about Agentic AI.

## Concepts

- [[concepts/agent2agent-protocol|Agent2Agent Protocol]] — Open protocol for capability discovery and stateful task coordination among agents.
- [[concepts/agent-evaluation|Agent Evaluation]] — Layered evaluation of routing, trajectories, outcomes, safety, efficiency, and adaptation.
- [[concepts/agent-harnesses|Agent Harnesses]] — Context, tools, state, controls, evaluation, and operations surrounding an agent model.
- [[concepts/agent-interoperability|Agent Interoperability]] — Protocol boundaries for tools, agents, interfaces, and commerce systems.
- [[concepts/agent-orchestration|Agent Orchestration]] — Control patterns for fixed workflows, single-agent loops, managers, and decentralized handoffs.
- [[concepts/agent-safety-and-control|Agent Safety and Control]] — Layered guardrails, runtime boundaries, observability, and human escalation.
- [[concepts/agent-skills|Agent Skills]] — Portable procedural packages with progressive disclosure, evaluation, and lifecycle governance.
- [[concepts/agent-tools|Agent Tools]] — Data, action, and orchestration interfaces designed for reliable, risk-aware model use.
- [[concepts/agentic-engineering|Agentic Engineering]] — Disciplined software development with agent harnesses, specifications, tests, and oversight.
- [[concepts/ai-agents|AI Agents]] — Definitions, core structure, use-case fit, costs, and evidence status.
- [[concepts/llm-wiki-pattern|LLM Wiki Pattern]] — Persistent agent-maintained synthesis built from curated sources, a wiki, and a governing schema.
- [[concepts/model-context-protocol|Model Context Protocol]] — Shared client–server boundary for connecting AI applications to data and tools.
- [[concepts/open-knowledge-format|Open Knowledge Format]] — Portable Markdown and YAML representation for exchanging curated knowledge bundles.
- [[concepts/react|ReAct]] — A reasoning–action–observation trajectory format for grounded agent behavior.
- [[concepts/spec-driven-development|Spec-Driven Development]] — Versioned specifications as shared contracts for humans, agents, and evaluations.
